# Screenshot Placeholders

Drop your app screenshots into this folder with these exact filenames:

| File | Screen | Recommended size |
|------|--------|-----------------|
| `welcome.png` | Welcome / Onboarding | 390×844 |
| `login.png` | Login Screen | 390×844 |
| `signup.png` | Sign Up Screen | 390×844 |
| `home.png` | Home Dashboard | 390×844 |
| `navigation.png` | Trail Navigation (full map) | 390×844 |
| `events.png` | Events Feed | 390×844 |
| `transport.png` | Lifts & Transport | 390×844 |
| `profile.png` | Profile Screen | 390×844 |
| `map-detail.png` | Trail map close-up (for Feature section) | any landscape |

## Tips

- PNG or JPG both work.
- The portfolio handles missing images gracefully — colored placeholder boxes show until you add the files.
- For best results, take screenshots at 390×844 (standard iPhone size) from the running app at http://localhost:5173.
- Crop out the browser chrome — just the app content.
